# Tran The Khoi - Personal Portfolio

## Overview
This repository contains the source code for my personal portfolio website. The website is designed to showcase my academic background, research publications, practical projects, and technical skills in the field of Artificial Intelligence, with a specific focus on Computer Vision and Large Language Models (LLMs).

## Website Structure
The portfolio is organized into the following sections to provide a comprehensive view of my professional profile:

- Snapshot: A quick summary of my current career direction and role as an AI Product Developer.
- About: Detailed information about my background, including my Bachelor's degree in Computer Science from the Industrial University of Ho Chi Minh City (IUH).
- Research: Highlights my graduation thesis and published scientific paper (SSRC) on "Safety Helmet Detection" utilizing the YOLOv8 architecture.
- Experience: Real-world work experience, including my time handling data labeling pipelines as a Data Annotator at Alchera Vietnam.
- Projects: Personal and academic projects I have developed, focusing on AI applications and problem-solving.
- Skills: A list of my technical stack, including Python, PyTorch, OpenCV, and fundamental web development technologies.
- Contact: Contact section for professional connections and opportunities.

## Technologies Used
This website is developed as a lightweight static page optimized for fast loading and smooth navigation, using:
- HTML5
- CSS3 (Including custom fixed header offset for seamless anchor navigation)
- Vanilla JavaScript

## Local Setup Instructions
To run and view this website locally on your machine, please follow these steps:

1. Clone the repository:
   git clone https://github.com/khoithetran/Khoi-Tran-Portfolio.git

2. Navigate to the project directory:
   cd Khoi-Tran-Portfolio

3. Open the `index.html` file in any web browser, or use an extension like Live Server in Visual Studio Code for the best viewing experience.
